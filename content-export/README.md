# Контент-архив Никиты Пичугина

Это независимый от текущего сайта музейно структурированный архив. Он разделяет произведения, цифровые файлы и места их показа на старом WordPress-сайте. Выдуманная авторская или музейная нумерация не используется; `workId` служит только внутренней технической связью.

## Что открывать

- `reports/review.html` — визуальная проверка записей.
- `exports/public-catalog.json` — готовые данные для будущего сайта; технические имена файлов не используются как заголовки.
- `owner-input.csv` — только сведения, которые невозможно достоверно восстановить с сайта.
- `data/works.jsonl` — музейно структурированные карточки произведений.
- `data/assets.jsonl` — метаданные оригинальных файлов.
- `data/placements.jsonl` — исходные разделы и порядок показа.
- `exports/lido.xml` — обменный файл LIDO 1.1.

## Медиа

- `media/originals/` содержит сохранённые исходники без перекодирования.
- `media/previews/` содержит отдельные WebP-превью для просмотра.
